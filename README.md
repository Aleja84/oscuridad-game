# OSCURIDAD — Horror Game

Juego de terror en primera persona desarrollado con Three.js. Explora una mansión oscura, recoge llaves y baterías, y encuentra la salida antes de que las criaturas te atrapen.

## Jugar

Abre `index.html` en el navegador o accede a la versión online en GitHub Pages.

---

## Historia

Despiertas en una sala oscura sin saber cómo llegaste ahí. La única salida está bloqueada. Tendrás que explorar habitaciones cerradas, recoger baterías para activar la caja eléctrica y escapar... pero no estás solo.

---

## Controles

| Tecla | Acción |
|-------|--------|
| `W A S D` | Moverse |
| Ratón | Mirar alrededor |
| `E` | Interactuar con objetos y puertas |
| `F` | Encender / apagar linterna |
| `B` | Mostrar / ocultar FPS |

---

## Objetivo

1. Explora la **Sala Central** y encuentra las **3 llaves** escondidas por el mapa
2. Usa las llaves para abrir el **Laboratorio**, la **Cocina** y la **Biblioteca**
3. Recoge las **3 baterías** repartidas por las habitaciones
4. Activa la **caja eléctrica** con las baterías
5. Escapa por la **puerta final**

---

## Habitaciones

| Habitación | Descripción |
|------------|-------------|
| Sala Central | Punto de inicio. Conecta con el resto del mapa |
| Laboratorio | Cerrada con llave. Contiene objetos útiles |
| Cocina | Cerrada con llave. Vigila las esquinas |
| Biblioteca | Cerrada con llave. Llena de secretos |
| Sótano | Accesible desde el inicio. Oscuro y peligroso |
| Cuarto Secreto | Esconde el **aturdidor**, útil contra enemigos |
| Morgue | No todo lo que hay ahí está muerto |
| Sala Final | La salida... si llegas |

---

## Enemigos

- **Fantasma** — Aparece en los pasillos. No se puede matar, solo huir
- **Stalker** — Te persigue activamente si te ve. Rápido y agresivo
- **Crawler** — Se arrastra por el suelo. Difícil de detectar en la oscuridad

Usa el **aturdidor** (recógelo en el Cuarto Secreto) para paralizar enemigos temporalmente.
